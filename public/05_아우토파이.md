## 1.3 AutoPi Cloud | Custom Code, Hooks, Workers  
Device 연결이 완료되었다면,  
이제 우리가 차량에서 AutoPi로 받아올 데이터를 처리하기 위해  
웹페이지에 코드 등록과 기타 작업들을 해줘야한다.  

순서는 **Custom Code 등록 > Hooks 등록 > Workers 등록** 이다.
